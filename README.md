# Weather Application:  
## About:  
 Weather application is an application that uses fetch API to display weather details based on the searched city   
## Built With:  
1- HTML5  
2- CSS3  
3- Boostrap V5.0  
4- Javascript using OOP and AJAX with Fetch API
